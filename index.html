Typografia

Czcionki i fonty

Pismo

Pismo to umowny system znaków służący do przedstawiania mowy. Pismo składa się ze znaków. Znaki nie mają określonego kształtu, ale posiadają ogólną strukturę. Przykładem pisma jest pismo łacińskie.

Krój pisma

Krój pisma składa się z glifów przedstawiających znaki pisma. Glify mają ściśle określone kształty. Przykładem kroju pisma jest Comic Sans lub Times New Roman.

Czcionka

Czcionka to metalowy prostopadłościan dawniej używany w drukarstwie do odbijania na papierze danego glifu pochodzącego z kroju pisma.

Font

Font to plik służący do używania kroju pisma na komputerze. Nie należy nazywać fontów czcionkami. Fonty mogą przedstawiać jedną z odmian danego kroju ( np. pogrubioną ). Przykładem fontu jest Arial.ttf lub Roboto-bold.ttf.

Budowa liter

Wydłużenie górne ( Ascender )

Element litery który wychodzi ponad linię średnią.

Wydłużenie dolne ( Descender )

Element który wychodzi poniżej linii podstawowej

Ligatura

Ligatura to graficzne połączenie dwóch znaków. Jest używana do poprawienia czytelności lub w celach estetycznych.

fi fl ffi ffl

//domyślne ligatury:

font-variant-ligatures: normal;

fi fl ffi ff

//całkowity brak ligatur:

font-variant-ligatures: none;

Podział linii tekstu

Linia podstawowa ( Baseline )

Wysokość na której wizualnie osadzone są litery

Linia średnia ( Mean line )

Linia określająca wysokość małych liter

wysokość x ( X-height )

Odległość między linią bazową a linią środkową. Przy dobieraniu czcionki używanej przy małym tekście warto wybrać czcionkę z większą wysokością x, aby poprawić czytelność.

Wysokość wersalików ( Cap height )

Linia określająca maksymalną graficzną wysokość wersalików ( dużych liter ).

Linia górna ( Ascender height )

Wydłużenia małych liter często sięgają wyżej niż duże litery. Linia górna określa ich wysokość.

Odstępy

Interlinia ( Leading )

Odległość pomiędzy liniami bazowymi dwóch linijek tekstu.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Quae sapiente animi similique necessitatibus velit voluptates quas. Consectetur suscipit veritatis impedit natus!

line-height: 1.5;

Lorem ipsum dolor sit amet consectetur adipisicing elit. Quae sapiente animi similique necessitatibus velit voluptates quas. Consectetur suscipit veritatis impedit natus!

line-height: 1;

Kerning

Odstęp pomiędzy poszczególnymi literami

VA, WA.

//domyślny dla czcionki kerning

font-kerning: normal;

VA, WA.

//kerning czcionki wyłączony

font-kerning: none;

Tracking ( Letter spacing )

Równy odstęp pomiędzy wszystkimi literami

wróżbita

/*domyślny tracking*/

letter-spacing: normal;

wróżbita

/*zmieniony tracking*/

letter-spacing: 20px;
